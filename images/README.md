## Map Banner Images

For the main map selection page, place images with these exact filenames:

| File | Map |
|------|-----|
| `terminus.jpg` | Terminus (BO6) |
| `liberty-falls.jpg` | Liberty Falls (BO6) |
| `citadelle.jpg` | Citadelle des Morts (BO6) |
| `der-eisendrache.jpg` | Der Eisendrache (BO3) |
| `shadows-of-evil.jpg` | Shadows of Evil (BO3) |
| `the-giant.jpg` | The Giant (BO3) |
| `gorod-krovi.jpg` | Gorod Krovi (BO3) |

**Recommended size:** 600×190 px (landscape). JPG or WebP works best.

If an image file is missing, the card falls back to its themed gradient background automatically.

## Usage

These step images can be referenced in the HTML guide pages to provide visual guidance for players. The images are sourced from the Steam Workshop guide by Masadaeus.
